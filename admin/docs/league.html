Each league page must have a json data file with the following information as well as a HTML page containing only {{> league}} in the saved document, nothing else. In addition to the following attributes the page data file must also include a "title": "" and "page-category": {} information like all other pages. Images for the league (including banners and map thumbnails) must be located at assets/img/tournament/ with map images in a folder called pool.
Main map attributes
Attribute Description Value Default
"name": The name of the league. String
"id": The ID used to reference the league's assets. String
Host Where the league tournament was hosted. This is defined in the page's "page-category": and can either be "overcast": true or "avicus": true.
"anounncement": The page which contains the league details. URL
"challonge": URL for the Challonge bracket. URL
"method": Tournament style. String
"selection": The amount of teams selected for the league and the method of selection (if appropriate). String
"players": The range of players per team roster permitted. String
"dates": [] The dates in which each stage took place. Stage info
"pool": [] The maps which were played during the league tournament. Map info
"fisrt": The name of the team that finished first. String
"second": The name of the team that finished second. String
"third": The name of the team that finished third. String
"teams": [] The teams which participated in the league tournament. Team info
Stage Information
Provides information about the different stages of the league tournament.
Attribute Description Value Default
"stage": The name of the stage being played. String
"date": The starting date for this specific stage. String
Map Pool Information
Provides information about the different maps that were played during the league tournament. Image thumbnails for these maps must be put into a folder called pool inside another folder which is the same as the league ID. These images must have the same name as the slug reference ID of the map.
Attribute Description Value Default
"map": The name of the map. String
"slug": The unique ID for referencing this map in other places of the website, including the map image. String
"gamemode": The map's gamemode. String
"round": The round in which this map was played. If the map was played in more than one round, create another map for that round. String
Team Information
Provides information about the teams that participated in the league tournament and the group stages.
Attribute Description Value Default
"group": The name of the group which the teams have been put into - eg. Group A. String
"participants": [] The teams which are included in this group.
"team": The name of the team. String
