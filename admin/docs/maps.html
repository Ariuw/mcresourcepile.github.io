Every map must be defined through the following data attributes in order to be displayed on the MapStack section of the website. The following table gives you an understanding on how to format the data for each map and what is required from each attribute. Maps must be ordered in alphabetical order within the data file.

Overcast maps listing: src/data/mapstack/overcast.json
Overcast unofficial maps listing: src/data/mapstack/overcast.json
Avicus maps listing: src/data/mapstack/avicus.json

Main map attributes
Attribute Description Value Default
"map": The name of the map. String
"slug": The map's unique slug ID, which is generally auto-generated by the associated network and is used to reference external map pages, downloads and XML documents. String
"authors": [] The author(s) of the map. User Data
"tags": [] The tags which describe the map's game mode and other features. Tag Data
"private": Disables all map links and prevents downloading from the site. Boolean false
"discussion": The map's discussion thread. URL false
"sourced": If the map was retrieved from the map's discussion thread or an alternate location which the map author(s) published. Boolean false
"commercial": Allows for commercial use of the map Boolean true
"download": If the map has a download located in a ResourcePile map repository Boolean true
"xml": If the map has a XML located in a ResourcePile map repository Boolean true
"cardinal": If the map works with the latest version of Cardinal. Working maps must be placed in the cardinal ResourcePile repository. Boolean false
Map authors
Provides information about who created the map and is displayed on map thumbnails and in the pre-download messages. Only the author name is required, however to reduce the amount of work required to keep names up-to-date, it is recommended to also provide the player UUID in addition to the player's display name.
Attribute Description Value Default
"player_name": The player's current display name. String
"player_uuid": The player's UUID, which can be found at mcuuid.net String
