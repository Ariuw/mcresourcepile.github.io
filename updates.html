Welcome to ResourcePile
Posted by ResourcePile 02-01-2017
Welcome to ResourcePile, a community driven project with contributors from Overcast Network and Avicus Network groups. This project offers a range of resources and tools related to these networks which can be broken down into three main groups:
  • StatTrack - Tracking various statistics such as longest kill distances by bow, and other website statistics like objective leaderboards
  • MapStack - A collection of publicly downloadable maps from Overcast Network and Avicus Network that players can use on their own servers or privately explore in single player
  • ToolRack - A collection of various other handy tools
